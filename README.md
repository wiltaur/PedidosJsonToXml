# Proyecto NET 8 C# - Convierte Json a XML y Viceversa

**Desarrollado con**:
- Mejores prácticas basadas en algunos principios SOLID y patrones de diseño ( Polly Resilience with Retry, Fluent Validations and Middlewares).
- Pruebas Unitarias con XUnit y Moq

**Despliegue en contenedor de Docker**:
- En el archivo **Dockerfile** de este proyecto se encuentran los pasos para la compilación en Docker, si se tiene instalado Docker con Linux para probar no hay inconveniente.
